# Лабораторная работа №11

## Задания:

### Задание 1
Про виртуальное окружение. Настройте виртуальное окружение и репозиторий.

### Задание 2
Парсим новости. Запустите `task_2/main.py`.

### Задание 3
Сохраняем в JSON. Запустите `task_3/main.py`.

### Задание 4
Создаём HTML страницу. Запустите `task_4/main.py`.

## Как запустить:

1. Скачайте все файлы в одну папку.
2. Запускайте файлы по порядку:
   - task_1/main.py
   - task_2/main.py  
   - task_3/main.py
   - task_4/main.py
3. Откройте файл `index.html` в браузере и увидите результат.
